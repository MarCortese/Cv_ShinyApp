# Curriculum Vitae
Creazione di un Curriculum vitae interattivo attraverso lo sviluppo di una shiny App.

Il curriculum si divide in

* Informazioni iniziali
* Formazione
* Esperienze lavorative
* Attestati
* Skills
* Ulteriori informazioni
